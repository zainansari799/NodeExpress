1. npm install tsd
this command is used to install all definition files.

link: http://definitelytyped.org/tsd/

2. tsd query node --action install

this command install typings folder in your project, where node.d.ts file is exists


3. tsd install jquery

this command generate jquery folder where jquery file exists
Note: Jquery folder generated in typings folder

4. npm install gulp --save-dev
e.g npm install filename --save-dev

this command install gulp in your project and also add devdependencies

5. npm install express --save
e.g npm install filename --save

this command install express in your project and also add dependencies

6. npm install body-parser
e.g npm install filename

this command is used for install core modules.
